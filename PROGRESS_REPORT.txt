PROJECT OVERVIEW
--------------------------------------------------------------------------------
Ledger AI is an intelligent financial management system designed to help users
track expenses, manage budgets, and gain insights into their spending patterns.
The application combines modern web technologies with AI/ML capabilities for
features like receipt scanning and financial forecasting.

================================================================================
                              TECH STACK
================================================================================

BACKEND:
  - Django 4.2+ with Django REST Framework
  - PostgreSQL database support (psycopg2-binary)
  - Celery + Redis for asynchronous task processing
  - Cloudinary for cloud-based image storage
  - Tesseract OCR for receipt text extraction
  - PyTorch + Transformers for ML capabilities
  - Prophet + scikit-learn for forecasting

FRONTEND:
  - React 19 with TypeScript
  - Vite as the build tool
  - React Router v7 for navigation
  - Recharts for data visualization
  - Axios for API communication
  - Tesseract.js for client-side OCR
  - Google Gemini API for generative AI features
  - Lucide React for icons

================================================================================
                           DATABASE SCHEMA
================================================================================

TRANSACTION MODEL:
  - id: Primary Key (auto)
  - owner: Foreign Key to User
  - title: CharField (max 200)
  - amount: DecimalField (12 digits, 2 decimal places)
  - date: DateField
  - category: CharField (max 100, optional)
  - notes: TextField (optional)
  - receipt_image: ImageField (optional, stored on Cloudinary)
  - created_at: DateTimeField (auto)
  - updated_at: DateTimeField (auto)

================================================================================
                           API ENDPOINTS
================================================================================

AUTHENTICATION:
  POST /api/register/          - Create new user account
  POST /api/login/             - Authenticate user
  POST /api/logout/            - End user session
  GET  /api/current-user/      - Get authenticated user info

TRANSACTIONS:
  GET    /api/transactions/           - List user's transactions
  POST   /api/transactions/           - Create new transaction
  GET    /api/transactions/{id}/      - Get transaction details
  PUT    /api/transactions/{id}/      - Update transaction
  DELETE /api/transactions/{id}/      - Delete transaction

UTILITIES:
  POST /api/upload-receipt/    - Upload and OCR parse a receipt
  GET  /api/export/            - Export transactions as CSV
  POST /api/import/            - Import transactions from CSV
